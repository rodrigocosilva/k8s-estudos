# Lab de Observabilidade Kubernetes

Stack de observabilidade rodando em Kubernetes no Mac Mini via OrbStack.

## Componentes
- Prometheus
- Grafana
- Jaeger
- Zabbix
- OpenTelemetry Collector
- App demo instrumentada

## Arquivos principais
- `lab-observabilidade-v2.yaml`
- `otel-collector.yaml`
- `demo-frontend.yaml`

## Objetivo
Estudar observabilidade moderna com métricas, tracing e monitoramento clássico.
